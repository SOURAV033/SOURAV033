### 🔥 [Customer Churn Prediction](https://github.com/SOURAV033/customer-churn-prediction)
> End-to-end ML pipeline + TensorFlow/Keras Deep Neural Network

- Built on **7,043 telecom customer records** with 25.7% churn rate
- **4 ML models** (Logistic Regression, Random Forest, Gradient Boosting, SVM) + **DNN**
- **5 engineered features** from first principles — tenure lifecycle, contract risk, bundle flags
- Best model: **ROC-AUC 0.690, Recall 65.8%** — identifies 2 in 3 churners
- **Production-grade modular Python**: preprocess.py → eda.py → train.py → predict.py
- Full visualisation suite: EDA panels, ROC curves, confusion matrices, feature importance

`Python` `TensorFlow` `scikit-learn` `SQL` `Power BI` `Pandas` `NumPy`
